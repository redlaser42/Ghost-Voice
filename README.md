# Jelly Voices
---
This is a mod that adds various voices to SPT. Over time as i find good clips I'll flush out the voices more. Do let me know if there are any problems also the voices are a bit loud. I get all my voice lines from recordings of the games their from or archived streams.

## Voice Request
You can leave a suggestion for a voice to add in the comments.
Requirements:
- Name
- What there from
- A video link to voice lines, with little background noise (optional)

## Installing
Just extract the folder and drop it in the main SPT directory. Make sure to extract using winrar or 7zip.

## Uninstalling
Before uninstalling make sure you change your voice back to a default. Then you can safely delete the "**JellyVoices**" folder in your "**SPT/user/mods**".
