Installing:
Just extract the folder and drop it in the main SPT directory. Make sure to extract using winrar or 7zip.

Uninstalling: 
Before uninstalling make sure you change your voice back to a default. Then you can safely delete the "Ghost Voice" folder in your "SPT/user/mods".

See modding guide: https://github.com/AvienReactor/JellyVoices/wiki/How-to-make-a-SPT-voice-mod
